# ESG Data Credibility — Chipotle Case

## Problem
Sustainability reporting relies heavily on modeled and estimated data, while consumer-facing tools imply a level of precision that is not fully supported.

## Risk
- Regulatory scrutiny on environmental claims  
- Erosion of ESG credibility  
- Weak audit defensibility  

## Key Insight
There is a structural gap between data quality (modeled vs primary) and how sustainability impacts are communicated to consumers.

## Solution
- Data tiering (primary vs modeled vs estimated)  
- Confidence labeling for all reported metrics  
- Claim governance rules aligned with data quality
